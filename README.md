# 概要
2人プレイのアクションゲームです。  
落ちてくるノーツやプレイヤー、出現ボスを倒してポイントを獲得し、ポイントが多いほうが勝ちです。

# 注意
このゲームはコントローラ入力のみ対応しています。  
2台のコントローラを接続した状態で起動してください。  
コントローラの入力形式はXInputのみ対応しています。
